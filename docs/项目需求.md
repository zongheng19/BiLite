# 类b站本地视频播放器
你需要制作一个与b站外观、交互相似的本地高性能、轻量化视频播放器。在子文件夹中，已经分析了b站前端播放器的界面和功能，你可以进行参考。你也应参考开源项目，并在需要的时候直接使用他们，比如FFmpeg、ffplayer、mpv等等。
## 具体需求如下
### 技术需求
- 播放器的GUI界面比较精美，争取与剔除了不必要功能的b站播放器一致。
- 支持常用的视频类型，解码性能较高。
- 客户端比较轻量化，不要使用例如Electron体积很大的方案。
- 具体技术栈有你确定，如果缺失环境或依赖，自行配置。
- 参考优秀的开源项目，如果有必要，可以直接使用。
### 交互需求
- 用户开箱即用，点击视频文件，立即打开并进行播放。
- 实现b站播放器的暂停、倍速播放、时间点跳转、播放时长记忆、外挂字幕等功能。
